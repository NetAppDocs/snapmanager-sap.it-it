= Arresto del server host SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Una volta terminato l'utilizzo di SnapManager, potrebbe essere necessario arrestare il server.

. Per arrestare il server, immettere il seguente comando, come utente root:
+
`smsap_server stop`
