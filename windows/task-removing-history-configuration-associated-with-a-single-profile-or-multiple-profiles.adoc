:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager consente di rimuovere le impostazioni della cronologia di un'operazione SnapManager. Questa operazione rimuove tutte le informazioni della cronologia associate a un singolo profilo o a pi첫 profili.

. Per rimuovere la cronologia delle operazioni SnapManager associate a uno o pi첫 profili, immettere il seguente comando:
+
`smsap history remove -profile {-name, profile_name [profile_name1, profile_name2] | all-repository-login [-passwordrepo_password] -usernamerepo_username-dbnamerepo_dbname-hostrepo_host-portrepo_port} -operation {-operationsoperation_name [operation_name1, operation_name2] | -all} [-quiet | -verbose]`
