= Il comando sspap schedule list
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Questo comando elenca le operazioni pianificate associate a un profilo.



== Sintassi

[listing]
----

smsap schedule list
-profile profile_name
[-quiet | -verbose]
----
